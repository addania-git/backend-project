# Dodger Game

## Overview
Dodger Game is a simple 2D browser game built using **HTML5 Canvas**, **CSS**, and **JavaScript**. The objective is to avoid falling obstacles for as long as possible. Your score increases the longer you survive.

---

## How to Play
- Use **Arrow Keys** or **A/D** to move left and right.
- Avoid the falling red blocks.
- The game ends when you collide with an obstacle.
- Click **Restart** to play again.

---

## Features
- Player-controlled movement
- Random obstacle generation
- Collision detection
- Score tracking
- Restart functionality
- Increasing difficulty over time

---

## How to Run
1. Download all files:
   - `index.html`
   - `style.css`
   - `game.js`
2. Place them in the same folder.
3. Open `index.html` in any modern web browser.
4. Enjoy the game!

---

## Technologies Used
- HTML5 Canvas API
- CSS for styling
- JavaScript for game logic
